# EduSexual PT

Plataforma de educação sexual em português europeu para jovens.

## Objetivo

Um espaço seguro e anónimo para jovens tirarem dúvidas sobre sexualidade, testarem conhecimentos e acederem a recursos confiáveis em Portugal.

## Funcionalidades

- **Artigos educacionais** sobre anatomia, contracepção, IST, relações e mais
- **Quiz interativo** para testar conhecimentos
- **Perguntas frequentes (FAQ)** com respostas a dúvidas comuns
- **Sistema de perguntas anónimas** para tirar dúvidas
- **Recursos em Portugal** - linhas de apoio e serviços de saúde

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS

## Como executar localmente

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev
```

O site estará disponível em http://localhost:3000

## Deploy

Este projeto está configurado para deploy automático no Vercel.

## Fontes

Informação baseada em fontes oficiais:
- APF (Associação para o Planeamento da Família)
- DGS (Direção-Geral da Saúde)
- DGE (Direção-Geral da Educação)
- OMS (Organização Mundial de Saúde)

## Aviso

Este site fornece informação educacional geral. Para aconselhamento médico personalizado, consulta um profissional de saúde.
