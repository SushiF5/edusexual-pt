# SPEC.md - EduSexual PT

## 1. Project Overview

- **Nome**: EduSexual PT
- **Tipo**: Webapp educacional anónimo
- **Objetivo**: Plataforma de educação sexual em português europeu para jovens tirarem dúvidas e testarem conhecimentos
- **Público-alvo**: Jovens (12-25 anos), pais, educadores
- **_stack**: Next.js 14, TypeScript, Tailwind CSS, Supabase (opcional para匿名 perguntas)

## 2. UI/UX Specification

### Layout
- Header fixo com navegação
- Homepage com introdução e categorias
- Página de artigos por tema
- Página de quizzes
- Página de perguntas anónimas
- Footer com recursos úteis

### Cores
- Primary: `#2D5A5A` (verde escuro profissional)
- Secondary: `#F4A261` (laranja acolhedor)
- Background: `#FAFAF9` (creme claro)
- Text: `#1F2937` (cinza escuro)
- Accent: `#E9C46A` (amarelo suave)

### Tipografia
- Headings: `Outfit` (moderno, legível)
- Body: `Source Sans 3` (otimizado para leitura)

### Componentes
- Cards de artigos com hover
- Accordion para FAQ
- Quiz com feedback imediato
- Formulário anónimo para perguntas

## 3. Funcionalidades

### Temas do Conteúdo
1. Anatomia e fisiologia
2. Contraceção e métodos preventivos
3. Infeções sexualmente transmissíveis (IST)
4. Relações e afetos
5. Orientação sexual e identidade de género
6. Consentimento e limites
7. Gravidez e planeamento familiar
8. Recursos em Portugal

### Quiz
- Perguntas por categoria
- Feedback imediato após cada resposta
- Resultado final com sugestões de melhoria
- Sem registo, sem guardar dados

### Tira Dúvidas (Anónimo)
- Formulário para submeter perguntas
- Perguntas frequentes pré-definidas
- Links para recursos externos (APF, DGS, etc.)

## 4. Conteúdo (PT-PT)
